MVP Scope

한 달 안에 보일 기능과 나중에 붙일 기능을 분리합니다.

라이브 요구사항은 시청, 검색, 방송 운영, 라이브 전용 채팅, 투표, 채팅창 분리, 시청자 수, 후원, 팔로우, 실시간 통계가 핵심입니다.

필수

라이브 기본 흐름

방송 입장, 검색, 시청자 화면, 라이브 전용 채팅, 채팅창 분리를 우선합니다.

필수

크리에이터 운영

방송 운영, 투표 생성, 채팅 설정, 보안 설정, 후원 설정, 실시간 통계를 분리합니다.

필수

팔로우와 후원

크리에이터 팔로우, Toss Payments 기반 충전, 후원 보내기, Alert 표시, 내역 확인을 MVP에 포함합니다.

선택

채널 확장

공개 채널, VOD, 매니저 기능은 발표 범위에 맞춰 후속으로 둡니다.

MVP 필수 기능

라이브 기능이라고 말하기 위해 반드시 보여야 하는 범위입니다.

기능연결 라우터접근비고
라이브 목록/live공개Anon User도 현재 방송 중인 목록을 볼 수 있습니다.
라이브 검색/live/search공개검색어 요약, 라이브 결과, 크리에이터 결과를 분리합니다.
라이브 시청/live/[creatorId]공개 읽기채팅 읽기와 방송 시청은 공개, 입력과 참여는 로그인 필요입니다. 입장 안내와 참여 제한 안내는 채팅 로그가 아니라 현재 사용자용 UI 컴포넌트로 표시합니다.
채팅창 분리/live/[creatorId]/chat공개 읽기방송 운영 중 별도 창으로 띄우는 팝아웃입니다. OBS용 overlay URL은 보안 설정에서 별도로 관리합니다.
라이브 전용 채팅/live/[creatorId]입력은 로그인 필요일반 채팅방과 별도 모델로 새로 설계합니다.
라이브 투표/channel/live, /live/[creatorId]참여는 로그인 필요크리에이터는 방송 중 투표를 생성·종료하고, 시청자는 라이브 화면에서 참여하며 결과를 실시간으로 확인합니다.
방송 운영/channel/live본인만방송 제목, 태그, 시작과 종료, 방송 중 빠른 설정을 처리합니다. 공개/비공개 설정은 MVP에서 제외합니다.
채팅 설정/channel/chat본인만채팅 입력은 로그인 유저만 가능하며, 참여 범위는 모든 로그인 유저 또는 팔로워로 제한합니다. 팔로우 후 대기 시간, 첫 채팅 전 규칙 확인 안내문, 저속 모드, 링크 차단, 금칙어를 관리합니다.
방송 보안/channel/security본인만스트림키, OBS 채팅창 URL, OBS 후원 Alert URL을 재확인 Dialog 기반으로 관리합니다.
크리에이터 후원 설정/channel/donation본인만채팅 후원 수신 조건, 후원 수익 요약, 데모용 정산 정보 저장, Alert·TTS 동작을 관리합니다.
실시간 통계/channel/analytics본인만방송 중 시청자, 채팅, 후원, 팔로우 증가를 확인합니다.
팔로우/user/follows, /live/[creatorId]로그인 필요대상은 크리에이터이며 공개 채널 게시판 기능과 분리합니다.
후원/live/[creatorId], /user/donations, /live/[creatorId]/alerts/donation로그인 필요보내기, 충전 Dialog, 채팅 강조 표시, OBS Alert overlay까지 MVP에 포함합니다.
채팅방 검색/chat/search로그인 필요기존 채팅 도메인의 검색 화면입니다.

선택 기능

MVP 이후 기능은 라우터 예약과 데이터 구조만 고려합니다.

기능라우터 후보접근비고
공개 채널 홈/channel/[creatorId]선택 기능SOOP 방송국 같은 팬카페형 페이지로 확장할 때 사용합니다.
공개 게시판/channel/[creatorId]/board선택 기능채널 기능 자체가 확정된 뒤 추가합니다.
다시보기/channel/[creatorId]/vod선택 기능방송 종료 후 자동 저장 옵션과 연결합니다.
클립/channel/[creatorId]/clips선택 기능방송 일부를 짧게 저장하고 공유하는 후속 콘텐츠 기능입니다.
구독후속 정책 확정 후 결정선택 기능크리에이터 후원형 멤버십 후보이며 결제, 혜택, 환불 정책을 먼저 정해야 합니다.
카테고리후속 정책 확정 후 결정선택 기능라이브 탐색 분류 기능입니다. 현재 MVP는 방송 태그 기반 검색을 우선합니다.
알림후속 정책 확정 후 결정선택 기능팔로우한 크리에이터의 방송 시작, 후원, 시스템 이벤트를 알려주는 후보입니다.
매니저/channel/permissions선택 기능MVP에서는 크리에이터 본인만 방송 운영합니다.
시청자 제재 관리/channel/viewers선택 기능강퇴·제한 이력은 후속 관리 화면으로 분리합니다.
라이브 채팅 사용자 액션후속 정책 확정 후 결정선택 기능닉네임 Popover, 메시지 신고, 사용자 차단, 구독권 선물은 MVP 이후 안전 정책과 결제 정책을 정한 뒤 검토합니다.
