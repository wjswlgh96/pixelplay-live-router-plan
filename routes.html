MVP Router Scope

라이브 중심 MVP에 필요한 라우터만 먼저 고정합니다.

공개 시청, 검색, 크리에이터 방송 운영, 투표, 후원, 팔로우를 MVP에 포함하고 공개 채널 게시판, VOD, 매니저 권한은 후속 후보로 둡니다.

Public

시청자 진입

랜딩, 라이브 목록, 라이브 검색, 라이브 시청, 채팅 팝아웃은 공개 라우터입니다.

Creator

방송 관리

방송 운영, 채팅 설정, 보안 설정, 실시간 통계를 분리합니다.

User

내 정보

프로필, 팔로우, 후원은 사용자 사이드바에서 같은 구조로 관리합니다.

Chat

기존 채팅 유지

일반 채팅방은 라이브 채팅과 별도 도메인으로 유지합니다.

MVP 필수 라우터

이번 단계에서 실제 네비게이션 연결 대상으로 보는 범위입니다.

라우터역할접근query비고
/공개 랜딩.공개없음로그인 CTA 기본 next는 /live입니다.
/live현재 송출 중인 라이브 목록.공개없음시청자 입장 UI입니다.
/live/search라이브 방송과 크리에이터 검색.공개?query=검색어치지직 검색처럼 라이브 결과와 크리에이터 결과를 분리합니다.
/live/[creatorId]특정 크리에이터의 현재 라이브 시청.공개 읽기없음채팅 입력, 팔로우, 후원, 투표 참여는 로그인 필요이며 투표 결과는 실시간으로 표시합니다.
/live/[creatorId]/chat라이브 채팅 팝아웃 보조창.공개 읽기없음방송 운영 화면의 채팅창 분리용입니다. OBS용 overlay URL은 /channel/security에서 토큰 포함 주소로 관리합니다.
/channel/live방송 운영.본인만없음방송 제목, 태그, 공개 상태, 송출 상태, 방송 중 투표 생성과 종료를 관리합니다.
/channel/chat채팅 설정과 금칙어.본인만없음채팅 참여 범위, 본인인증, 저속 모드, 금칙어를 관리합니다.
/channel/security방송 보안 설정.본인만없음스트림키, OBS 채팅창 URL, 선택사항인 후원 Alert URL 후보를 관리합니다.
/channel/analytics실시간 통계.본인만없음시청자 수, 채팅량, 후원, 팔로우 증가를 확인합니다.
/chat채팅방 목록.로그인 필요필터 후보일반 채팅 도메인입니다.
/chat/search채팅방 검색.로그인 필요?query=검색어제목 결과와 닉네임 결과를 분리합니다.
/chat/room/[roomId]채팅방 상세.로그인 필요없음기존 채팅방 상세의 새 경로입니다.
/user/profile내 프로필 설정.로그인 필요없음/user 기본 이동 대상입니다.
/user/follows팔로우한 크리에이터 목록.로그인 필요필터 후보MVP에 포함합니다.
/user/donations후원 홈.로그인 필요?tab=... 후보잔액, 충전 Dialog, 충전·사용·보낸 후원 내역을 한 화면에서 처리합니다.
/api/payments/toss/confirmToss 결제 승인 API.서버 처리없음후원금 충전 완료 후 호출합니다.
/api/payments/toss/webhookToss 결제 웹훅 API.서버 처리없음결제 상태 보정에 사용합니다.

후속 후보

현재 MVP에서는 라우터 파일을 먼저 만들지 않습니다.

기능라우터 후보판단이유
채널 관리 홈/channel후속방송 운영과 설정이 먼저입니다.
리허설 방송/channel/rehearsal후속정식 송출 전 테스트 방송은 선택 기능입니다.
공개 채널 홈/channel/[creatorId]후속SOOP 방송국 같은 팬카페형 페이지는 MVP 이후 검토합니다.
공개 게시판/channel/[creatorId]/board후속공개 채널 홈이 확정된 뒤 연결하는 선택 기능입니다.
콘텐츠 관리/channel/content후속VOD와 다시보기 관리는 방송 종료 후 기능입니다.
클립/channel/[creatorId]/clips후속방송 일부를 짧게 저장하고 공유하는 선택 기능입니다.
구독후속 정책 확정 후 라우터 결정후속크리에이터 후원형 멤버십 후보이며 결제와 혜택 정책이 먼저 필요합니다.
카테고리후속 정책 확정 후 라우터 결정후속라이브 탐색 분류 후보입니다. MVP는 방송 태그 기반 검색을 우선합니다.
알림후속 정책 확정 후 라우터 결정후속팔로우한 크리에이터의 방송 시작, 후원, 시스템 이벤트 알림 후보입니다.
OBS 후원 Alert/live/[creatorId]/alerts/donation후속후원 수신 시 OBS Browser Source에 알림 오버레이를 띄우는 선택 기능입니다.
라이브 채팅 사용자 액션후속 정책 확정 후 라우터 결정후속닉네임 Popover, 메시지 신고, 사용자 차단, 구독권 선물은 안전·결제 정책 확정 후 검토합니다.
권한 관리/channel/permissions후속MVP 방송 운영은 크리에이터 본인만 수행합니다.
