MVP Scope

한 달 안에 보일 기능과 나중에 붙일 기능을 분리합니다.

라이브 요구사항은 시청, 검색, 방송 운영, 라이브 전용 채팅, 채팅창 분리, 투표, 시청자 수, 후원, 팔로우, 실시간 통계가 핵심입니다.

필수

라이브 기본 흐름

방송 입장, 검색, 시청자 화면, 라이브 전용 채팅, 채팅창 분리를 우선합니다.

필수

크리에이터 운영

방송 운영, 채팅 설정, 보안 설정, 실시간 통계를 분리합니다.

필수

팔로우와 후원

크리에이터 팔로우, Toss Payments 기반 충전, 후원 보내기, 내역 확인을 MVP에 포함합니다.

선택

채널 확장

공개 채널, VOD, 매니저 기능은 발표 범위에 맞춰 후속으로 둡니다.

MVP 필수 기능

라이브 기능이라고 말하기 위해 반드시 보여야 하는 범위입니다.

기능연결 라우터접근비고
라이브 목록/live공개Anon User도 현재 방송 중인 목록을 볼 수 있습니다.
라이브 검색/live/search공개검색어 요약, 라이브 결과, 크리에이터 결과를 분리합니다.
라이브 시청/live/[creatorId]공개 읽기채팅 읽기와 방송 시청은 공개, 입력과 참여는 로그인 필요입니다.
채팅창 분리/live/[creatorId]/chat공개 읽기OBS Studio 채팅창 URL로도 활용합니다.
라이브 전용 채팅/live/[creatorId]입력은 로그인 필요일반 채팅방과 별도 모델로 새로 설계합니다.
방송 운영/channel/live본인만방송 정보, 빠른 설정, 송출 상태, 공지, 투표를 처리합니다.
채팅 설정/channel/chat본인만채팅 권한, 본인인증, 모드, 금칙어를 관리합니다.
방송 보안/channel/security본인만스트림키와 OBS 채팅창 URL을 재확인 Dialog 기반으로 관리합니다.
실시간 통계/channel/analytics본인만방송 중 시청자, 채팅, 후원, 투표 참여율을 확인합니다.
팔로우/user/follows, /live/[creatorId]로그인 필요대상은 크리에이터이며 공개 채널 게시판 기능과 분리합니다.
투표/live/[creatorId], /channel/live참여는 로그인 필요방송자는 생성, 시청자는 참여, 결과는 실시간 반영합니다.
후원/live/[creatorId], /user/donations로그인 필요보내기와 충전 Dialog까지 MVP에 포함합니다.
채팅방 검색/chat/search로그인 필요기존 채팅 도메인의 검색 화면입니다.

선택 기능

MVP 이후 기능은 라우터 예약과 데이터 구조만 고려합니다.

기능라우터 후보접근비고
공개 채널 홈/channel/[creatorId]선택 기능SOOP 방송국 같은 팬카페형 페이지로 확장할 때 사용합니다.
공개 게시판/channel/[creatorId]/board선택 기능채널 기능 자체가 확정된 뒤 추가합니다.
다시보기/channel/[creatorId]/vod선택 기능방송 종료 후 자동 저장 옵션과 연결합니다.
매니저/channel/permissions선택 기능MVP에서는 크리에이터 본인만 방송 운영합니다.
시청자 제재 관리/channel/viewers선택 기능강퇴·제한 이력은 후속 관리 화면으로 분리합니다.
