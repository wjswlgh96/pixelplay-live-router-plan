Chat Room Enhancement Notes

채팅방 고도화 후보는 간단한 백로그로 보존합니다.

라이브 기능을 먼저 개발하고, 프로젝트 기간이 남았을 때 채팅방 도메인에 추가할 수 있는 후보를 큰 틀과 구현 방향만 간략히 정리합니다.

고도화 후보라이브 이후 참고할 채팅 백로그

입력 보정, 메시지 탐색, 권한, 비밀방, 공지, 알림 후보를 한 문서에 모읍니다.

메시지 탐색, 권한과 운영, 후속 연동 후보로 나누어 이후 작업 범위를 파악합니다.

입력과 메시지 조회

입력 안정성과 과거 메시지 접근 방식을 다룹니다.

macOS 한글 입력 중복

마지막 조합 글자가 한 번 더 입력되는 문제를 보정합니다. composition 이벤트, isComposing, keyCode === 229, 전송 snapshot을 함께 확인합니다.

IME guardsubmit snapshot

메시지 백업과 조회 범위

기본 조회 범위는 last_joined_at 이후로 제한하고, 과거 메시지는 백업 기능에서 날짜 범위를 선택해 불러오는 방향을 고려합니다.

last_joined_atdate range

날짜 선택 이동

date picker 또는 날짜 system message를 통해 특정 날짜 주변 메시지로 이동합니다. 데이터는 날짜 기준 cursor pagination으로 불러오는 방향을 고려합니다.

date pickercursor pagination

권한과 관리 기능

메시지 관리와 방 운영 권한을 확장하는 후보입니다.

메시지 수정과 삭제

본인 메시지 수정과 삭제, 방장 또는 매니저의 악성 메시지 삭제를 고려합니다. 수정 표시는 기존 message.modified_at 컬럼을 활용합니다.

modified_athard delete 후보

채팅 매니저 권한

chat_room_member.role 컬럼으로 owner, manager, member를 구분합니다. 실제 권한 검증은 RPC에서 처리합니다.

ownermanagermember

강퇴와 ban 정책

현재처럼 강퇴된 유저는 해당 방에 다시 입장할 수 없는 정책을 유지합니다. 내부 구현은 기존 is_banned 흐름을 유지합니다.

is_banned재입장 불가

방 운영 UX

비밀방, 검색, 공지, 공유 흐름처럼 채팅방 사용성을 보강하는 후보입니다.

  • 채팅방 비밀번호. 비밀번호는 hash로 저장하고, 참여 Dialog에서 비밀번호 입력 후 입장할 수 있게 합니다. 목록에는 비밀방 표시와 필터를 고려합니다.
  • 메시지 검색. Ctrl + F와 비슷하게 검색어 입력, 현재 결과 index, 전체 결과 수, 위아래 이동 버튼을 제공합니다. 검색 버튼은 나가기 popover 근처에 배치하는 방향이 자연스럽습니다.
  • 공지 메시지. chat_room_notice 테이블로 현재 활성 공지와 이전 공지 목록을 관리합니다. 상단에는 일부만 표시하고 전체 보기는 Dialog로 제공합니다.
  • 초대 링크와 public preview. preview 안에서 로그인 CTA 또는 로그인 컴포넌트를 보여주는 방향을 고려합니다. 비밀방은 실제 입장 시 비밀번호 검증을 거칩니다.

후속 연동 후보

알림 체계가 준비되면 함께 검토할 수 있는 기능입니다.

메시지 신고

신고 테이블, 신고 상태, 관리자 확인 UI를 추가하는 방향으로 고려합니다. 알림 시스템이 준비되면 함께 검토합니다.

방별 알림 설정

chat_room_member.notification_enabled 컬럼을 후보로 둡니다. 브라우저 알림 또는 앱 푸시 기능과 함께 검토합니다.

후속 연동

메시지 신고, 방별 알림, public preview 개선은 라이브 이후 필요에 따라 확장합니다.

reportnotification

라이브 기획 연결 메모

채팅방 고도화는 라이브 기능 개발 이후 남은 기간에 진행할 후보입니다.

  • 라이브 우선. 현재 우선순위는 라이브 기능 feature입니다.
  • 채팅방 고도화는 백로그. 이 문서는 라이브 개발 이후 채팅방 기능을 보강할 때 참고합니다.
  • 도메인 분리 유지. 라이브 채팅은 일반 채팅방과 UX가 다르므로 별도 도메인으로 설계하는 방향을 유지합니다.

이 문서는 채팅방 고도화 후보를 간략히 보존하기 위한 탭 문서입니다.
