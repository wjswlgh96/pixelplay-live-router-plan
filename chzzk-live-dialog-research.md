# 치지직 라이브 Dialog와 스튜디오 조사

조사 일시: 2026-05-26.
조사 도구: Chrome 확장 탭 claim 후 DOM과 안전한 클릭으로 확인.
민감 정보 원칙: 스트림키 값은 표시하지 않았고, 재발급·삭제·방송 시작 같은 상태 변경 액션은 실행하지 않았습니다.

## 조사한 탭

- `리그오브레전드 - CHZZK`.
- `치지직 - CHZZK`.
- `고뇌하는 형 5929의 치지직 스튜디오`.
- `PixelPlay - RGB`.

## 라이브 검색 결과

- 검색 입력 placeholder는 `채널, 라이브, 동영상 검색`이고 검색어는 유지됩니다.
- 결과는 라이브 방송 카드와 크리에이터 카드가 함께 노출됩니다.
- 라이브 결과는 `/live/{id}`로 이동하고, 카드 안에 LIVE 상태, 시청자 수, 방송 제목, 크리에이터, 카테고리와 태그가 보입니다.
- 크리에이터 결과는 채널 링크, 인증 표시, 팔로워 수, 팔로우 버튼을 함께 보여줍니다.
- 더보기 버튼이 섹션 단위로 존재합니다.

## 라이브 시청 화면

- 라이브 화면은 플레이어, 방송 정보, 팔로우, 구독, 구독 선물, 더보기 메뉴, 클립 목록, 채팅 패널로 구성됩니다.
- 채팅 패널에는 채팅 접기, 더보기 메뉴, 채팅 메시지 목록, 프로필 팝업, 이모티콘, 후원하기, 채팅 입력이 있습니다.
- 방송 정보 더보기 메뉴에서는 공유하기와 신고하기가 Popover 형태로 보였습니다.
- 후원하기를 클릭했을 때 현재 계정 상태에서는 채팅 참여 동의 팝업이 먼저 보였습니다.
- PixelPlay는 후원 자체는 별도 Dialog로 설계하되, 비로그인 또는 미동의 상태에서는 로그인 또는 참여 동의 흐름을 먼저 태웁니다.

## 스튜디오와 방송 운영

- 스튜디오 좌측 메뉴는 대시보드, 방송 관리, 분석, 시청자 관리, 콘텐츠 관리, 수익 관리, 채널/권한 관리, 내정보, 공지사항으로 구성됩니다.
- PixelPlay MVP는 전체 메뉴를 따라가지 않고 방송 운영, 채팅 설정, 보안 설정, 실시간 통계로 줄입니다.
- 방송하기 화면은 스트리밍 소프트웨어 연결 안내, OFF 상태, 스트림 상태, 실시간 통계 요약, 방송 제목, 카테고리, 태그, 미리보기 이미지, 연령 제한, 시청 국가, 유료 프로모션, 클립 만들기, 다시보기 설정, 업데이트 버튼을 보여줍니다.
- 빠른 설정에는 채팅 이용 권한, 이모티콘 모드, 저속 모드, 알림 설정, 금칙어 설정이 보입니다.
- 스트리밍 설정 안내 버튼은 Dialog를 열고 OBS와 PRISM 다운로드, 스트림 URL, 스트림 키, 복사 버튼, 확인 버튼을 보여줍니다.

## 방송 설정과 금칙어

- 설정 화면에는 스트림 URL, 스트림 키, 복사, 표시, 재발급 버튼이 있습니다.
- 채팅 참여 범위는 모든 시청자, 팔로워 전용, 운영자 전용으로 보입니다.
- 본인인증 여부, 이모티콘 모드, 저속 모드, 채팅 규칙, 금칙어 관리가 같은 설정 화면에 있습니다.
- 금칙어 관리는 Dialog가 아니라 관리 페이지 형태로 이동했습니다.
- PixelPlay는 정보량을 줄이기 위해 스트림 URL과 스트림 키는 `/channel/security`, 채팅 정책은 `/channel/chat`로 분리합니다.

## PixelPlay Dialog 적용 기준

- 로그인 필요 액션: 채팅 입력, 팔로우, 후원은 로그인 안내 Dialog로 보냅니다.
- 후원하기: 금액, 메시지, 보유 포인트, 취소, 후원 보내기 버튼을 포함합니다.
- 후원금 충전: `/user/donations` 내부 Dialog로 두고 Toss 결제 흐름으로 연결합니다.
- 스트림키 표시: 재확인 Dialog 후 표시합니다.
- 스트림키 재발급: 위험 액션으로 danger 톤의 재확인 Dialog를 사용합니다.
- OBS 채팅창 URL 복사: 큰 Dialog보다 복사 안내 Popover가 적합합니다.
- 금칙어 추가: 관리 패널 안에서 입력과 목록을 같이 보여줍니다.
- 설정 저장 전 이동: 변경 사항이 있을 때만 확인 Dialog를 띄웁니다.
